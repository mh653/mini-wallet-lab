# MINI WALLET Lab

## 概要

Python（Flask）とMySQLを使用してショッピングサイトのデモを制作しました。
商品を選択して購入できるEC機能と、管理者として商品管理や注文管理ができる管理画面を実装しています。

PythonでのWebアプリ開発の基礎理解を目的として作成しました。

---

## 主な機能

### ユーザ機能

・商品一覧表示
・商品詳細表示
・カート機能（Session管理）
・商品購入機能
・ユーザログイン機能
・検索機能

### 管理者機能

・管理者ログイン機能
・注文情報確認
・注文ステータス更新
・商品登録
・商品編集

## 工夫した点

・Blueprintを使用して機能を分離しました
・カート情報をSessionで管理することでログイン無しでも購入可能にしました
・管理画面から商品のCRUD操作ができるようにしました
・画像アップロード機能を実装しました
